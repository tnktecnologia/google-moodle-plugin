Google SAML Authorization
=========================

SSO plugin between Moodle and Google Apps using SAML.

Features
--------

- GSAML Authentication mode is added on users administration panel.

Requirements
------------

Moodle 2.x.

Installation
------------

Simply install the plugin and add the block to a course page.
