Google Data Block
=================

This block is focused primarily on the Google Apps Provisioning API and syncing Moodle user accounts to Google.

Features
--------

- Moodle to Google synchronization per each insert & delete database event.

Requirements
------------

Moodle 2.x.

Installation
------------

Simply install the plugin and add the block to a course page.
